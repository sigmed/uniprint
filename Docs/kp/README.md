## Если у вас 17 минут

Прочитайте **`00-executive-summary.md`** + **`03-pricing.md`** — этого достаточно для решения «брать / не брать».

## Если у вас 60 минут

Прочитайте всё по порядку — `00 → 01 → 02 → 03 → 04 → 05`.

## Если у вас 5 минут

Только Executive Summary (`00`):
- **Stage A** (4 кабинета, ~4 мес, **7-9 млн ₽**) — запуск производства + склад + менеджер.
- **Stage B** (+~6 нед, **2-3 млн ₽**) — клиентский портал + кабинет учредителя + автодокументы.
- **Phase 2** (~2 мес, **4-5 млн ₽** T&M) — Face Control + сдельная ЗП + финансы.
- **Phase 3** (~1 мес, **1.5-2 млн ₽** T&M) — multi-warehouse + BI + ЭДО.
- **Total ownership cost:** **15-19 млн ₽** за ~9-10 мес.

---

## Что вам нужно сделать после прочтения

1. **Прочитайте** хотя бы `00` + `03` (~17 мин).
2. **Подтвердите принципиальное согласие** на обсуждение контракта.
3. **Ответьте на 5 ключевых вопросов** до 29 мая 2026 — это разблокирует старт работ:
   - Q1: Юрисдикция / юр. форма / СНО.
   - Q2: Vendor для Face Control (можно отложить до 1 ноября 2026).
   - Q3: Хостинг (Yandex.Cloud / другое).
   - Q4: Миграция данных из старой системы — нужна или нет.
   - Q5: Эквайринг + ОФД (можно отложить до 1 декабря 2026, не блокирует Stage A).

   Подробно — в `04-risks.md` § 1.

4. **Запросите встречу** для обсуждения деталей и подписания контракта.
