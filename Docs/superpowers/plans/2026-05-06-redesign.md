# UniPrint Redesign 2026-05-06 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Заменить визуальный слой прототипа (`prototype/`) на дизайн, подтверждённый владельцем 2026-05-06 (warm cream `#FAF6EF` + coral `#D9531E` + Fraunces serif headings + JetBrains Mono для ID/цен + Manrope body + dark sidebar + PWA phone-frame для mobile + status-pulse + BR-callouts inline).

**Architecture:** Forward-only refactor визуального слоя. **Tech Stack:** Next.js 16 + React 19 + TS + Tailwind v4 (`@theme` syntax) + shadcn/ui (как база) + lucide-react + Fraunces/Manrope/JetBrains Mono через `next/font/google` + MSW для моков + Playwright для e2e + Biome для линта.

**Design source-of-truth:**

**Rule A (Context7) — MANDATORY before any code:** Перед каждым шагом, который трогает Tailwind v4 / Next.js 16 / shadcn/ui / lucide-react / framer-motion / next/font — запрос в Context7 (`mcp__plugin_context7_context7__resolve-library-id` → `query-docs`). Это в DoD каждой Task.
